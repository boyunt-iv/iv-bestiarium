About the Project

IV Bestiarium: The Hidden Ark is operated as an official project workspace for publishing original creative content. The application is limited to authorized use by the project owner or approved collaborators and is not offered as a public social media management platform.

Original Content

The project publishes creative materials, captions, descriptions, and related media prepared by or for IV Bestiarium: The Hidden Ark.

Controlled Access

Connected social accounts must be owned, controlled, or explicitly authorized by the person using the application.

Limited Data Use

Platform data is processed only as needed to authenticate accounts, publish requested content, maintain security, and provide support.

Publishing Workflow

The application helps manage a narrow publishing workflow for authorized accounts. It is designed to respect platform requirements and keep account access limited to the permissions needed for the requested function.

  • Authorize a connected platform account through the platform's approved OAuth flow.
  • Prepare original project media, captions, descriptions, hashtags, and publishing instructions.
  • Submit content to TikTok or another connected platform through approved API functionality.
  • Review upload status, publishing responses, and operational error messages when available.
  • Disconnect account access or request deletion by contacting the project support email.
The application does not sell personal data, operate a public advertising network, or use TikTok data for profiling, data brokerage, or unrelated third-party marketing.
